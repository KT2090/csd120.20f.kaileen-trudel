Acronym Glossary

HTML
HyperText Markup Language
HTTP
HyperText Transfer Protocol (protocol that defines how clients and servers communicate)
IP Address
Internet Protocol Address (unique address assigned to a device connected to the internet)
DNS
Domain Name Server (translate domain names into IP addresses)
TCP/IP
Transmission Control Protocol/Internet Protocol
